Rainbow Six Siege Companion

An aplication I created using Android Studio and Flutter

When I was playing Rainbow Six Siege, I wanted to improve my performance with my team. We often lost a lot of time teaching each team member the strategies they needed to apply. Therefore, I created this app which provides strategies to follow, including which operator to play, which positions to reinforce, which bans to apply, and which positions to hold.

The app ultimately provides all this information if the creator adds all the necessary images. We developed strategies for a few maps, but when the game was updated to include new mechanics, such as a pool of reinforcements instead of 2 per operator, the app became less relevant, and I stopped update it.
