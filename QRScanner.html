QRScanner

App created in a team of 2 during the Android Studio class at UQAC

During the Android Studio class, we had to create an Android application to learn how to use Kotlin and various technologies. Our team decided to create a mobile app for reading QR codes. We used Google's API to allow the user to log in or sign up with their Google account.

Additionally, the user can choose to stay logged in. We decided to keep the application simple because we believed that such features could be reused in other projects. By mastering these technologies, we will be more comfortable re-implementing them in future applications.
