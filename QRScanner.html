QRScanner

App created in team of 2 in the android studio class of the UQAC

During the class of Android Studio, we had to create an android application to learn how to use Kotlin and some technologies. Our team decided to creat a mobile app of QR code reading. We used google's API, to ask the user to load or sing on with his google account.

Moreover, the user can chose to stay connected. We decided to make an application so simple, because we thought that such features could be reused on other projects. Thanks to this mastery of these technologies, we will be more comfortable reimplementing them on other applications.
