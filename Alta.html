Alta

Game made entierly in Java

During my two years in IUT, I joined the Paris Descartes "Enigma" video game design club, which still exists today. Eager to progress in java, we embarked on this gigantic project to create a topdown rogue lite RPG with java FX. Very busy with our school projects and our courses, we worked sporadically on Alta.

The project ended with the creation of the game engine, the procedural generation of places, an AI with a pathfinding system. Real-time battles and Lvl Up. We also created in parallel a map editor, still in Java to help level designers.

The project ended with a good amount of featurres implemented, however, the lack of artist and sound designers left the game with a lot of placeholders. The choice of Java as the only language as well as the refusal to use a game engine slowed down the design, but made us all tremendous progress in Java and in object programming in general.

We also used FramaGit and developped with eclipse as IDE.
